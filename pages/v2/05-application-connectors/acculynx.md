## Account setup

You will be asked for the following values when installing the AccuLynx connector within an account:

| Value        | Description                           |
| :----------- | :------------------------------------ |
| **API Key**  | The API Key of your AccuLynx account. |
| **Base URL** | The Base URL of the API instance.     |
